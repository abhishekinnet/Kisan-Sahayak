# **App Name**: Kisan Sahayak

## Core Features:

- Multilingual Agronomy Assistance: Provide real-time, personalized agronomic guidance in the farmer's preferred language, using local units and farmer-friendly terms. LLM tool provides advice via text or speech; translation API to support all major languages.
- Disease Diagnosis: Diagnose plant diseases from images, providing likely disease names, confidence scores, and immediate actions for the farmer. Use LLM tool to include soil/weather information; recommend chemical or organic controls.
- Fertilizer and Nutrient Advisory: Offer fertilizer and nutrient advice based on soil tests, crop type, and weather. Recommend balanced schedules with organic alternatives and remedies for deficiencies.
- Weather Alert and Advisory: Deliver weather-based advisories with short warnings, practical steps to reduce risk, using LLM to suggest expert connect in severe conditions.
- Market Price Information: Provide market price insights by comparing local prices with state averages, suggesting selling decisions and storage tips.
- Live Expert Connect: Enable farmers to connect with agricultural experts via video, audio, or WhatsApp, summarizing the problem to enhance session efficiency.
- Financial Record and Farm Ledger: Maintain financial records, calculating profits/losses, providing money-saving tips, and updates on relevant schemes and news.

## Style Guidelines:

- Primary color: HSL(110, 65%, 50%) which is equivalent to a vibrant green (#40bf29), representing growth and health in agriculture.
- Background color: Desaturated green (#E0F7E0), creating a calm and natural feel.
- Accent color: Yellow-Green (HSL(80, 50%, 60%) which is equivalent to #A8D03A) provides contrast and highlights key actions.
- Design a clear, intuitive layout optimized for accessibility on small screens, focusing on readability.  The system will support Indian farmers (86% of the farming population) by overcoming digital literacy and language barriers.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet warm and accessible feel suitable for diverse literacy levels.
- Use simple, clear icons representing crops, weather, and financial aspects.
- Use subtle transitions and animations to provide feedback and guide the user through the app.
