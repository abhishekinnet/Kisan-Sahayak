.# 🌾 Kisan-Sahayak

Kisan-Sahayak is a smart digital platform designed to empower farmers with real-time information, expert guidance, and modern agricultural tools. The system provides crop recommendations, weather updates, government schemes, chatbot support, video calling with experts, and more — all in one place.

---

## 🚀 Features

- 📊 **Crop Recommendation System**  
- 🌤️ **Real-Time Weather Updates**  
- 🤖 **Smart Chatbot for Farmer Queries**  
- 🎥 **Live Video Calling With Agricultural Experts**  
- 🏛️ **Latest Government Schemes for Farmers**  
- 📰 **Daily Agricultural News**  
- 💧 **Irrigation & Fertilizer Suggestions**  
- 🛠️ **Simple & Farmer-Friendly UI**  
- 📱 **Fully Responsive Design for Mobile Users**

---

## 📥 Clone This Repository

Follow the steps below to clone and run the project locally.

### 1️⃣ Install Git (if not installed)



Open Terminal (VS Code / CMD / Git Bash)

Navigate to the folder where you want to clone the project:

cd Desktop


Clone the Repository
git clone https://github.com/your-username/Kisan-Sahayak.git

4️⃣ Open the Project Folder
cd Kisan-Sahayak

5️⃣ Install Dependencies (For Node.js / React projects)
npm install


6️⃣ Start Development Server
For Vite:
npm run dev

For Create React App:
npm start
